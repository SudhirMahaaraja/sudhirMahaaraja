## Projects

**Visitor Behavior Detection System**
Computer vision system for real-time foot traffic analysis, dwell time measurement, and engagement detection across video feeds. Achieved 92% detection accuracy using YOLO, Caffe, Amazon Rekognition, and Azure Vision Service. Video inputs stored in S3; results persisted in MongoDB.

**Virtual Health Assistant**
AI-powered disease prediction platform delivering diagnostic support with 85% accuracy. Designed for continuous availability as a digital health monitoring tool.

**Social Distancing Compliance Monitor**
Real-time video analysis system for social distancing enforcement built with YOLOv3 and OpenCV.

**DeepFace Recognition and Clustering**
GPU-accelerated face grouping system using VGG-Face embeddings and advanced clustering techniques for high-precision identity management.

**Tourist Behavior Analysis**
Data analytics study uncovering travel trends and consumer behavior patterns. Visualized with Power BI and published as peer-reviewed research.

**BERT-Based Multi-Label Text Classifier**
Multi-label text classification pipeline using BERT with Optuna-driven hyperparameter optimization for real-time document categorization.

**Audio-to-Text Transcription System**
High-accuracy speech transcription pipeline leveraging large language models for automated, scalable audio processing.
